## üìå Project Overview

This project is a simple To-Do List application built using **React Native** and **Expo**.

The application allows users to:

- Add new goals using `TextInput`
- Display goals using `FlatList`
- Delete goals with confirmation dialog
- Experience a modern Dark Blue UI design

The app demonstrates the use of React Native core components and state management using `useState`.
